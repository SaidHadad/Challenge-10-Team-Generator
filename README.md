  ## Getting Started

  ### Prerequisites

  Node, node_modules, express.js

  ### Installation

  npm install --y <br>
  npm install express <br>
  <br>
  In package.json add<br> 
  "start": "node server.js" to scripts<br>
  <br>
  start the server
